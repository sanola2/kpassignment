Kakaopay Assignment
---------------

open API를 사용해 이미지와 텍스트를 메인 화면에 List or Grid 형태로 draw하고, 특정 item 클릭 시 상세 내용 출력

날씨 API 사용
https://openweathermap.org


Specs
-------------
- Kotlin
- Minimum SDK 21
- MVVM Architecture

Libraries
--------------
- Retrofit2
- RxJava
- Glide
- DataBinding
- Android Architecture Component
